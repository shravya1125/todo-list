# 📝 To-Do List App

A simple To-Do List web app to manage your daily tasks efficiently.

## 🚀 Features
- 🖊️ Add, Edit, Delete Tasks
- 🎨 Responsive Design
- 💾 Stores Tasks in Local Storage
